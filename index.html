Introduction

CSS  -  Cascading Style Sheets - is the language for handling the look and feel of web pages. CSS handles how web pages appear to users, that is the presentation and accessibility of the document to users.

CSS can be used for basic styling of texts, to complex effects like animations. For example, CSS can be used to define color, font-size, and/or background of websites created in a markup language (HTML) document.

Prerequisites

To get the best of this documentation, you are expected to know the following -

  • HTML
  • How the web works

CSS Syntax

CSS syntax takes the form of a rule-based language. You define rules stating how styles should be applied to particular element(s) on a web page.

The syntax consist of selector(s), and property: value; pair which is referred to as declaration.


            selector {
              property: value;
            }

We can have more than one property: value; pair in a CSS rule. And this will we call a declaration block. Let’s assume that you want the heading of your web page to be of text blue and a font size of 6em. You will achieve this in CSS, writing the rule as a declaration block, like so.

           h1 {
            color: blue;
            font-size: 6em;
          }

Comments in CSS

Comments  are  non executable piece of code 

In CSS, comments can be written in two ways -

  • Single line comments like so /* This is a single line comment in CSS */
  • Multi-line comments like so 
    /* This is a single line 
          comment in CSS */

Applying CSS

The earlier part of this article focused on defining CSS and its syntax, but CSS is meant to work closely with markup documents like HTML. So how do we apply CSS to a web app - HTML document? 

  • Inline Style

    • A single HTML element can be styled, by applying CSS inside the style attribute of the HTML element.

              
            Red text styled in-line

  • Internal Style

    You apply CSS internally to an HTML document by putting the <style> element, in the <head> of the HTML document.

  • External Style

    You apply a stylesheet to an HTML document using the <link> element like this, 

    <link rel="stylesheet" href="styles.css">
    and this will be in the head of the HTML document.
